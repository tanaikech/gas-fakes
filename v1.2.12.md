- version of gas-fakes cli now aligns with gas-fakes and comes from shared package.json
- cli can now provide output from gas-fakes - example below

```
```bash
gas-fakes -a '{"key":"sample"}' -s 'const res = "The provided argument is " + args.key; return JSON.stringify({output: res});' | grep '^{"output":' | jq -r '.output'
```
