# gas-fakes v2.5.4 Release Notes

## Overview
Version 2.5.4 is a massive release that brings complete parity to the **XML Service**, expands **UrlFetchApp** options, drastically improves **SpreadsheetApp** grid and chart capabilities, and fundamentally refactors the **gf_agent** skills system for local documentation discovery.

## Key Features & Additions

### 1. XML Service Completion
- Achieved full, comprehensive parity for the `XmlService`.
- Implemented and rigorously tested all XML node types: `Cdata`, `Comment`, `Content`, `DocType`, `EntityRef`, `Text`, and `ProcessingInstruction`.
- Expanded `FakeElement` and `FakeDocument` handling.
- Implemented robust `FakeFormat` capabilities for XML serialization and output formatting.

### 2. UrlFetchApp Enhancements
- Significantly improved `UrlFetchApp.fetch()` to support a broader array of advanced fetch options and payload configurations.
- Integrated robust request/response testing in `testfetch.js` to ensure alignment with live GAS behaviors.

### 3. FakeSpreadsheet & Charts Enhancements
- **Themes**: Implemented full support for `FakeSpreadsheetTheme`, enabling custom fonts, colors, and the `getSpreadsheetTheme()`, `setSpreadsheetTheme()`, and `resetSpreadsheetTheme()` workflows.
- **Settings**: Added getter and setter support for `Locale`, `TimeZone`, and `IterativeCalculation` (cycles, thresholds, and interval properties) by correctly interfacing with the Spreadsheet metadata model.
- **Grid & View Adjustments**: Implemented `setFrozenRows`, `setFrozenColumns`, `unhideColumn`, and `unhideRow`.
- **UI Mocks**: Introduced stubbed UI methods like `toast`, `show`, and `updateMenu` on Spreadsheet directly to prevent script errors when interacting with UI bound scripts.
- **Row/Column and Sheet Proxies**: Fully implemented a wide range of Spreadsheet methods that proxy down to the Active Sheet (e.g. `appendRow`, `deleteRow`, `hideColumn`, `insertRowsAfter`, `autoResizeColumn`, etc.), and native sheet management shortcuts (`rename`, `deleteActiveSheet`, `duplicateActiveSheet`, `findSheetByName`).
- **FakeEmbeddedChartBuilder Styling**: Vastly expanded the Chart Builder fake to cover detailed text, style, and axis modifications. Implemented `setLegendTextStyle`, `setTitleTextStyle`, `setXAxisTextStyle`, `setYAxisTextStyle`, `setPointStyle`, `useLogScale`, `reverseCategories`, and more.
- **TestSheets**: Split into multiple files to handle the increased test coverage.

### 4. gf_agent Refactor & Discovery
- Completely overhauled the `gf_agent` architecture by deprecating individual service skill files (e.g., `drive.md`, `spreadsheet.md`, `document.md`).
- Introduced a powerful new centralized knowledge file: `17-local-documentation-discovery.md` to handle local codebase indexing and LLM skill discovery.

### 5. Documentation & Core Tools
- Massively updated `doccreation` analyzer tools (`gi-analyzer-all.js`, `gi-render.js`, etc.) to process the vast new XML and Spreadsheet progress payloads.
- Added enhancements to `ScriptApp` and `FakeAuthorizationInfo`.

## Bug Fixes
- **FakeSheet `appendRow` Bug Avoidance**: Fixed testing contexts where the proxy `appendRow` behavior would fail when testing against a fresh fake sheet containing exactly 1 row by accurately managing active sheet context and clearing.
- **MCP Configuration Updates**: Fixed an incorrectly configured file path in `.gemini/settings.json` which prevented the local `omlx` MCP server from properly exposing the `query_local_model` tool.
