## v2.2.5

### Security & Authentication Updates

- **Enhanced MS Graph Local Cache Security:** Transformed the Microsoft Graph local token cache from plaintext JSON (`.msgraph-token.json`) to a locally-signed JSON Web Token (`.msgraph-token.jwt`). This prevents casual viewing and tampering of the token data by the node process or accidental exposure of plain-text credentials.
- **Improved MS Graph Tenant Defaults:** Updated the default tenant for Azure CLI and Interactive SDK fallback login flows to prioritize `consumers`. This correctly targets personal Microsoft accounts, which is the most rigorously tested and primary path for the "Keyless" and "Silent Runtime" experiences, minimizing generic "SPO License" exceptions for external users and preventing stall states during background authentication. 
- **Updated Documentation:** Revised `msgraph.md` and `oddities.md` to document the `.msgraph-token.jwt` cache changes, its location in `.gitignore`, and the explicit `consumers` default tenant guidance.
